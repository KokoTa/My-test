cmd命令框下：

md xx 创建名为xx的文件夹
cd xx 选择名为xx的文件夹

npm -v
npm root                        当前包的安装路径
npm root -g                     全局包的安装路径
npm install xx                  安装命令
npm install xx -g               全局安装
npm init                        编写package.json 用于github
npm ls                          显示所有安装包
npm remove xx                   删除包
npm update xx			更新包
npm update			更新自己
